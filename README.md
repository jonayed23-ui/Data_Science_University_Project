# Legal Document Analysis System

## Project Overview

The Legal Document Analysis System is a Data Science project designed to analyze and process legal documents efficiently. The system uses machine learning and natural language processing (NLP) techniques to extract meaningful information from legal texts, helping users understand and organize legal documents more effectively.

## Features

* Legal document preprocessing
* Text cleaning and normalization
* Feature extraction from legal documents
* Machine learning-based document analysis
* Data visualization and insights
* Model evaluation and performance analysis

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## Installation

1. Clone the repository:

```
git clone https://github.com/jonayed23-ui/Data_Science_University_Project.git
```

2. Navigate to the project directory:

```
cd Data_Science_University_Project
```

3. Install dependencies:

```
pip install -r requirements.txt
```

## Usage

1. Open Jupyter Notebook:

```
jupyter notebook
```

2. Run the notebooks or Python scripts to perform data analysis and model training.
